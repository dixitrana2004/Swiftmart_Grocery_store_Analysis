# 🛒 Swiftmart Grocery — Sales Dashboard

An interactive Excel dashboard built on the Swiftmart Grocery dataset (8,524 transaction rows), uncovering sales patterns across outlet size, location tier, item type, and fat content — and challenging the assumption that big cities and large outlets drive quick-commerce success.

![Dashboard Snapshot](Dashboard_Snapshot.png)

## 📊 Key KPIs

| Metric | Value |
|---|---|
| Total Sales | **$1.20M+** |
| Total Items Sold | **8,523** |
| Average Sales per Transaction | **$141.0** |
| Average Customer Rating | **4.0** |

## 🔍 Key Insights

### Outlet Size Performance
- **Medium outlets** lead with **$507.9K** in sales (highest)
- Small outlets: $444.8K
- High outlets: $249.0K (lowest)

### Outlet Location Performance
- **Tier 3** cities are the top contributor at **$472.1K**
- Tier 2: $393.2K
- Tier 1: $336.4K

➡️ **Tier 3 cities + Medium-sized outlets are actually leading sales** — contrary to the common assumption that metros and large stores drive quick-commerce growth.

### Outlet Type Analysis
- **Supermarket Type 1** dominates with **$787.5K** in sales and **5,577 items** sold
- Grocery Store: $151.9K
- Supermarket Type 2: $131.5K
- Supermarket Type 3: $130.7K

### Product Insights
- **Regular Fat** items account for 65% of sales ($776.3K)
- **Low Fat** items account for 35% of sales ($425.4K)
- Top-selling categories: Fruits & Vegetables, Snack Foods, Household

### Sales Trend
- Sales show stable growth across years (2011–2022)
- **Peak year: 2018**, with sales reaching **$204.5K**

## 📈 Dashboard Components

| Visual | Purpose |
|---|---|
| Donut Charts (Fat Content, Outlet Size) | Visualize contribution % and compare segments |
| Bar Charts (Item Type, Outlet Type, Outlet Location) | Compare performance across categories and spot top contributors |
| Area/Line Chart (Outlet Establishment Trend) | Show how sales evolved over time and highlight peak years |
| Stacked Bar (Fat by Outlet) | Compare Low Fat vs. Regular sales across Tier 1, 2, and 3 outlets |
| Filter Panel | Slice data by Outlet Size, Outlet Location Type, and Item Type |

## 🛠️ Tools Used

## 💡 Final Takeaway

The next big opportunity in quick-commerce isn't just expansion — it's **expansion in the right locations**. Smaller cities (Tier 3) and medium-sized outlets are emerging as the real growth drivers, not the metro/large-format stores typically assumed to lead the market.
